사무실 PC를,
카페에서 폰으로 그대로

Tailscale로 내 기기들을 하나의 보안망으로 묶고, Sunshine·Moonlight로 고성능 PC 화면을 어디서든 저지연으로 조작합니다. 큰 파일도 클라우드 없이 기기끼리 직접. 설치부터 보안까지 9단계로 따라 하세요.

How it works

한눈에 보는 구조

Tailscale이 모든 기기를 하나의 보안망으로 깔아주고, 그 위에서 호스트 PC(Sunshine)의 화면이 내 기기(Moonlight)로 흐릅니다.

Tailscale — 모든 기기를 잇는 보안망 Sunshine — 화면을 송출하는 호스트(1대) Moonlight — 화면을 받아 조작하는 기기들

즉, Tailscale = 길, Sunshine = 보내는 쪽, Moonlight = 받는 쪽. 셋이 한 세트로 "내 PC를 손안으로" 옮겨 옵니다.

The three tools

세 도구가 각자 맡는 일

역할만 구분하면 헷갈릴 게 없습니다. 어디에 까는지가 핵심.

Tailscale

모든 기기에 설치

흩어진 기기를 하나의 암호화된 사설망으로 묶습니다. 위치·통신사·공유기 무관하게 서로 직접 연결 — 포트포워딩이나 고정 IP가 필요 없어요. 파일 전송(Taildrop)도 여기서.

Sunshine

조종당할 PC(호스트) 1대

내 PC를 '송출 호스트'로 바꿉니다. 화면·소리·입력을 GPU로 실시간 인코딩해 내보내요. 고성능 데스크탑 한 대에만 깔면 됩니다. (오픈소스)

Moonlight

조종에 쓸 기기들

그 화면을 받아 저지연으로 띄우는 뷰어입니다. 노트북·폰·태블릿·미니PC·TV 어디든 설치. 키보드·마우스·터치·게임패드·오디오까지 그대로 전달해요. (오픈소스)

Real use cases

어떤 기기끼리, 무엇을 할 수 있나

실제로 이렇게 씁니다. 화살표는 "조종하는 기기 → 조종당하는 PC" 방향이에요.

집·외근 노트북→사무실 워크스테이션

외근·재택에서 메인 PC 그대로

가벼운 노트북·폰만 들고 나가도, 사무실의 고성능 PC를 그대로 불러와 무거운 디자인·렌더링·편집 작업을 합니다. 현장 기기 성능과 무관.

폰→작업용 데스크탑

외부에서 렌더·생성 모니터링

ComfyUI 이미지/영상 생성이나 긴 렌더가 도는 동안, 밖에서 폰으로 진행 상황을 확인하고 다음 작업을 큐에 걸어둡니다. 자리 비워도 안심.

데스크탑↔노트북·폰

대용량 시안·영상 직접 전송

클라우드 업로드·다운로드 없이 Taildrop으로 기기끼리 바로 보냅니다. 수 GB 시안·영상도 사내망 속도로, 용량 제한·압축 걱정 없이.

회의실 TV·미니PC→작업 PC

큰 화면으로 리뷰·시연

거실·회의실의 TV나 미니PC에서 작업 PC 화면을 그대로 띄워, 팀과 함께 시안을 크게 보며 리뷰하거나 시연합니다. 파일 옮길 필요 없이.

Prerequisites

시작 전 준비물

세 가지면 충분합니다. 나머지는 진행하며 같이 깔아요.

조종당할 PC (호스트)

GPU 있는 데스크탑·워크스테이션 권장(NVIDIA·AMD·인텔 내장 모두 OK). 접속하려면 켜져 있어야 하니 절전은 끄기.

조종할 기기

노트북·폰·태블릿·미니PC·TV 등 화면 받을 기기. 여러 대 등록해도 됩니다.

로그인 계정 1개

Tailscale에 쓸 계정 하나(Google·Microsoft·GitHub 등). 모든 기기를 같은 계정으로 로그인하면 한 망으로 묶입니다.

Step by step

9단계로 완성하기

각 단계 끝의 체크박스를 누르면 진도가 저장됩니다. 끊겨도 다시 오면 그대로예요. 단계마다 어느 기기에서 하는 일인지 태그로 표시했습니다.

1
Tailscale · 모든 기기

Tailscale 설치하고 같은 계정으로 로그인

먼저 모든 기기를 한 보안망으로 묶습니다. 호스트 PC와 조종에 쓸 기기 전부에 설치하세요.

  • Windows·macOS·Linux: tailscale.com/download
  • Android·iPhone·iPad: 앱스토어/플레이스토어에서 Tailscale 검색
  • 설치 후 모든 기기를 같은 계정으로 로그인 (Google·Microsoft·GitHub 등 택1)
같은 계정으로 로그인한 기기끼리만 한 망(tailnet)이 됩니다. 회사·집 기기를 섞어 묶어도 OK.
성공 신호 — 각 기기의 Tailscale 트레이/앱이 Connected로 표시됨
2
Tailscale · 호스트

호스트 PC의 Tailscale 주소 확인

6단계에서 연결할 때 쓸 호스트 PC의 주소를 미리 적어 둡니다.

관리자 콘솔› Machines› 호스트 PC의 100.x 주소
  • login.tailscale.com/admin/machines 또는 트레이 메뉴에서 기기 목록 확인
  • 호스트 PC의 Tailscale IP(100.x.x.x) 또는 이름(MagicDNS)을 메모

호스트에서 자기 IP를 바로 확인하려면(선택):

tailscale ip -4
공유기 포트포워딩·고정 IP·DDNS 전혀 필요 없습니다. Tailscale이 알아서 뚫어줍니다.
성공 신호 — 호스트의 100.x.x.x 주소를 손에 넣음
3
Sunshine · 호스트

호스트 PC에 Sunshine 설치

조종당할 PC에만 설치합니다. 화면을 내보내는 쪽이에요.

  • github.com/LizardByte/Sunshine → Releases → 운영체제에 맞는 설치본(Windows .exe)
  • 설치 중 가상 디스플레이 드라이버·방화벽 허용 요청이 나오면 모두 허용
  • 설치되면 백그라운드 서비스로 자동 실행됩니다(트레이 아이콘)
Sunshine은 호스트(메인 PC) 한 대에만 깝니다. 조종에 쓸 노트북·폰에는 5단계의 Moonlight를 깔아요.
성공 신호 — 트레이에 Sunshine 아이콘이 보이고 서비스가 실행 중
4
Sunshine · 호스트

Sunshine 웹UI 초기 설정

호스트 PC의 브라우저에서 Sunshine 설정 화면을 엽니다.

https://localhost:47990
  • 자체 서명 인증서 경고가 뜨면 고급 → 계속 진행 (내 PC라 안전)
  • 최초 1회 관리자 아이디·비밀번호를 만듭니다 — 원격 접속의 관문이니 강력하게
  • 인코더가 자동 감지됩니다(NVIDIA=NVENC). Applications에 Desktop이 기본 등록돼 있는지 확인
Applications 목록에 Desktop이 있으면 전체 바탕화면을 스트리밍할 수 있습니다. 특정 앱만 띄우려면 여기서 추가도 가능.
성공 신호 — 웹UI 로그인 성공 + Applications에 Desktop 존재
5
Moonlight · 조종 기기

조종할 기기에 Moonlight 설치

이번엔 화면을 받을 기기 차례입니다. 노트북·폰·태블릿 등 원하는 만큼.

  • Windows·macOS·Linux: moonlight-stream.org
  • Android·iPhone·iPad·Apple TV: 스토어에서 Moonlight 검색
조종할 기기에도 Tailscale이 같은 계정으로 로그인돼 있어야 사무실 밖에서도 붙습니다(1단계).
성공 신호 — Moonlight 실행 시 "호스트 추가/검색" 화면이 보임
6
Moonlight + Sunshine

페어링 — Tailscale 주소로 연결

이제 조종 기기와 호스트를 짝지읍니다. 여기가 "어디서든"의 핵심이에요.

  1. Moonlight에서 호스트 수동 추가(+ 버튼) → 2단계의 호스트 Tailscale IP(100.x.x.x) 입력
  2. 호스트가 목록에 뜨면 클릭 → Moonlight가 4자리 PIN을 보여줍니다
  3. 호스트 PC의 Sunshine 웹UI(https://localhost:47990) → PIN 탭 → 그 코드 입력 → 저장
일반 IP(192.168.x.x) 말고 Tailscale IP(100.x.x.x)를 넣으세요. 그래야 사무실 밖에서도 똑같이 연결됩니다.
성공 신호 — Moonlight에 호스트의 Desktop 썸네일이 나타남
7
Moonlight · 조종 기기

원격 제어 시작 + 화질·지연 잡기

Moonlight에서 Desktop을 선택하면 호스트 화면이 그대로 뜹니다. 키보드·마우스·터치로 바로 조작하세요.

화질·지연 팁

  • 같은 사무실 망(LAN)이면 4K·60fps도 매끄럽습니다
  • 외부망·LTE는 해상도·비트레이트를 낮추세요 — 예: 1080p / 10~20Mbps / 60fps
  • 끊기면 fps를 60→30으로, 유선이나 5GHz Wi-Fi 사용
  • 스트림 종료 단축키(기본): Ctrl+Alt+Shift+Q
성공 신호 — 내 기기에서 호스트 바탕화면이 보이고 조작됨 🎉
8
Tailscale · 파일 공유

기기 간 파일 주고받기

원격 제어와 별개로, Tailscale 망에 묶인 기기끼리 파일을 바로 보냅니다. 클라우드를 거치지 않아요.

방법 A — Taildrop (가장 간편, 폰↔PC도)

  • 데스크탑: 트레이 Tailscale → Send file → 받을 기기 선택
  • 폰: 공유 시트에서 Tailscale 선택 → 기기 선택
# 보내기 (기기이름 뒤 콜론 ":" 주의)
tailscale file cp 시안.psd 집노트북:
# 받는 쪽에서 꺼내기
tailscale file get .

방법 B — 네트워크 드라이브(폴더 통째로)

  • 호스트에서 폴더 공유 후, 다른 기기에서 \\100.x.x.x\공유폴더로 접속·드라이브 매핑
성공 신호 — 보낸 파일이 다른 기기에 도착함
9
보안 · 필수

안전하게 잠그기

원격 접속은 강력한 만큼 잠금이 중요합니다. 아래는 빠뜨리면 안 되는 것들.

  • 포트포워딩 금지 — Sunshine을 공유기로 외부 개방하지 마세요. 접근은 오직 Tailscale 망으로만.
  • Sunshine 웹UI 비밀번호를 강력하게, 공유 금지(4단계)
  • Tailscale 관리자 콘솔의 ACL로 어떤 기기가 호스트에 닿을지 제한
  • 호스트 자리 비울 땐 화면 잠금, 키 만료/기기 승인 정기 점검
핵심 한 줄: Tailscale 안에서만 열고, 인터넷에는 절대 직접 열지 말 것.
성공 신호 — 외부에 열린 포트 0개, 허용된 기기만 호스트 접근

Troubleshooting

문제 해결 FAQ

대부분의 막힘은 아래 중 하나입니다.

Q1 사무실 밖에서 연결이 안 돼요
Moonlight에 일반 IP(192.168.x.x) 대신 Tailscale IP(100.x.x.x)를 넣었는지 확인하세요. 그리고 호스트·조종 기기 양쪽 Tailscale이 모두 Connected여야 합니다(같은 계정).
Q2 화면이 끊기거나 지연이 심해요
외부망에선 Moonlight 설정에서 해상도·비트레이트를 낮추세요(예 1080p·10~20Mbps). fps를 60→30으로, 가능하면 유선이나 5GHz Wi-Fi를 쓰면 크게 좋아집니다.
Q3 Sunshine 웹UI(47990)가 안 열려요
https://를 직접 붙여 입력하고(http 아님), 인증서 경고는 "계속 진행"으로 넘기세요. 그래도 안 되면 Sunshine 서비스 실행 여부와 방화벽을 확인합니다.
Q4 호스트 PC가 절전으로 꺼져 못 붙어요
접속하려면 호스트가 켜져 있어야 합니다. 전원 옵션에서 절전·최대 절전을 끄거나 시간을 늘리세요. (고급) Wake on LAN으로 깨우는 방법도 있습니다.
Q5 PIN을 넣었는데 페어링이 안 돼요
PIN은 호스트 PC의 Sunshine 웹UI → PIN 탭에 넣어야 합니다(조종 기기 아님). 시간이 지나면 PIN이 만료되니, Moonlight에서 다시 시도해 새 PIN으로 입력하세요.
Q6 소리·게임패드·여러 모니터도 되나요?
네. Moonlight가 오디오·게임패드·HDR까지 전달합니다. 다중 모니터는 Sunshine 설정에서 출력할 디스플레이를 고르거나, 가상 디스플레이로 단일 화면을 만들어 쓰면 깔끔합니다.
Security

꼭 지켜야 할 보안 수칙

원격 데스크탑은 곧 내 PC로 통하는 문입니다. 아래 네 가지는 필수예요.

인터넷에 직접 열지 말 것 (가장 중요)

Sunshine 포트를 공유기로 포트포워딩하면 전 세계에 노출됩니다. 접근은 오직 Tailscale 망으로만 — 그게 이 구성의 핵심 이점입니다.

Sunshine 비밀번호는 비밀번호급

웹UI 계정이 뚫리면 내 PC가 통째로 넘어갑니다. 길고 강력하게, 다른 곳과 재사용 금지, 공유 금지.

Tailscale ACL·기기 승인 관리

관리자 콘솔에서 ACL로 호스트에 닿을 수 있는 기기를 제한하고, 모르는 기기가 망에 들어오지 않게 기기 승인과 키 만료를 점검하세요.

호스트 화면 잠금·세션 정리

자리를 비울 땐 호스트를 화면 잠금하고, 쓰지 않는 페어링 기기는 Sunshine에서 삭제해 두세요.
